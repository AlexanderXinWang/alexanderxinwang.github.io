Gradient Descent for Linear Regression

For each step along the error curve:
$$
\textbf{w}^{t+1} = \textbf{w}^t - \alpha \nabla L(\textbf{w}^t) \
= \textbf{w}^t - \frac{\alpha}{n} \sum_{i=1}^n \nabla L_i( \textbf{w}^t, \textbf{x}i, y_i)

L(w, X, y) = \frac{1}{n} \sum{i=1}^n (y_i - \langle w, x_i \rangle)^2 \
= \frac{1}{n} ||y - Xw||^2
$$

Stochastic Gradient Descent

At each step, compute estimate of gradient using only one randomly selected sample, and move in the direction it indicates.

Many of the steps will be in the wrong direction, but progress towards minimum occurs on average, as long as the steps are small.

Bias

The difference between the true value and the mean prediction of the model

Variance

Variation of the model output if we train many models with a new sample of training data each time. (The level of the model output depends on the training data)
